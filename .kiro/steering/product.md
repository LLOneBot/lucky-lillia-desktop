# Lucky Lillia Desktop

A desktop management application for QQ bots built with Python and Flet framework.

## Purpose
- Manages PMHQ (QQ bot framework) and LLOneBot processes
- Provides a GUI for starting/stopping bot services
- Handles configuration, logging, and updates
- Supports system tray integration for background operation

## Target Platform
- Windows x64 only
- Requires administrator privileges for full functionality

## Key Features
- Process lifecycle management (PMHQ, LLOneBot, QQ)
- Real-time log collection and viewing
- Configuration management with JSON persistence
- Auto-update checking via NPM registry
- System tray with minimize-to-tray support
- Theme switching (light/dark)

## Language
- UI and documentation are in Chinese (Simplified)
- Code comments are in Chinese
