# 应用图标说明

## 图标要求

为了给应用添加自定义图标，请准备一个 `.ico` 格式的图标文件。

### 规格要求

- **格式**：ICO（Windows图标格式）
- **推荐尺寸**：256x256 像素
- **文件名**：`icon.ico`
- **位置**：项目根目录

### 图标内容建议

建议图标设计包含以下元素：
- QQ机器人相关的视觉元素
- 清晰的轮廓和对比度
- 适合在小尺寸下显示

### 在线转换工具

如果你有 PNG 或 JPG 格式的图片，可以使用以下在线工具转换为 ICO：

1. **IcoConverter**
   - 网址：https://www.icoconverter.com/
   - 支持多种尺寸

2. **Convertio**
   - 网址：https://convertio.co/zh/png-ico/
   - 简单易用

3. **Online-Convert**
   - 网址：https://www.online-convert.com/
   - 支持批量转换

### 使用方法

1. 准备或转换图标文件
2. 将文件命名为 `icon.ico`
3. 放置在项目根目录（与 `main.py` 同级）
4. 运行打包脚本 `build.bat`

### 无图标打包

如果不需要自定义图标，可以：

1. 编辑 `qq-bot-manager.spec` 文件
2. 找到 `icon='icon.ico'` 这一行
3. 删除或注释掉这一行
4. 打包时会使用 PyInstaller 的默认图标

## 图标示例

你可以参考以下风格设计图标：
- 机器人头像
- QQ企鹅 + 齿轮（表示管理）
- 聊天气泡 + 设置图标
- 简洁的字母标识（如 "QBM"）

## 注意事项

- ICO 文件可以包含多个尺寸（16x16, 32x32, 48x48, 256x256）
- 确保图标在小尺寸下仍然清晰可辨
- 避免使用过于复杂的细节
- 建议使用透明背景
